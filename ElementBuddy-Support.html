ElementBuddy

An interactive periodic table reference for iPhone and iPad — covering all 118 elements with color-coded tiles, rich detail cards, and a built-in quiz mode.

Overview

What is ElementBuddy?

Elements is a periodic table reference designed for students, science enthusiasts, and anyone curious about the building blocks of matter. Browse the complete table, explore every element in depth, and test your knowledge — all in one beautifully designed app.

Getting Started

First launch

When you open the app, you'll see the periodic table grid. Each tile shows an element's symbol, atomic number, and name, color-coded by its category. Tap any tile to open its detail card.

Main View

The Periodic Table Grid

Browsing the table

The full 18-column periodic table layout is displayed by default. On iPhone in portrait, elements appear in an adaptive flow grid. Rotate to landscape to see the classic periodic table shape. On iPad, the full table shape is always shown.

Tapping an element

Tap any element tile to open its detail card. Swipe down on the card, or tap the × button, to close it.

Filtering by category

Tap the filter icon in the toolbar to highlight a specific element category. The selected category stays bright while all other tiles are dimmed — the table shape is preserved so you can see exactly where in the table that category appears. Tap the same category again, or choose "All Elements," to clear the filter.

Searching

Tap the search bar at the top and type an element name, symbol, or atomic number. Matching elements stay highlighted; non-matching tiles are dimmed so the table shape stays intact.

Grid vs. list view

Tap the view toggle in the toolbar to switch between the grid and a scrollable list. In list mode, the category filter narrows results rather than just highlighting them.

On the detail card, tap the speaker icon to hear the element's name spoken aloud. Useful for learning correct pronunciations of less familiar element names.

Test Yourself

Quiz Mode

Tap the quiz button in the toolbar to open Quiz Mode. Choose from four question types:

Symbol → Name

Given an element's symbol, identify the element name.

Name → Symbol

Given an element's name, identify the correct chemical symbol.

Atomic Number

Match an element to its atomic number.

Category

Identify which category an element belongs to.

Session length

Before starting, choose how many questions you'd like: 5, 10, or 20. Your score is shown at the end of each session.

Tips for studying

  • Start with Symbol → Name if you're a beginner — it's the most visual question type.
  • Use Category quizzes to reinforce where elements sit in the table.
  • Return to the grid after a quiz to look up any elements you missed.

Data

Data Accuracy

All 118 elements and their properties — including atomic masses — have been reviewed against NIST SP 966 (June 2024), the U.S. National Institute of Standards and Technology's official periodic table publication.

If you spot a discrepancy in element data, please contact us through the App Store support link.
FAQ

Frequently Asked Questions

Why does the grid dim elements when I filter?

Dimming (rather than removing) non-matching elements preserves the periodic table's shape. The position of each element in the table is part of what the table communicates — hiding tiles would break that structure.

Why does the table look different in portrait vs. landscape on iPhone?

In portrait, the full 18-column table is too wide to display at a legible size, so elements appear in an adaptive flow grid. Rotate to landscape to see the classic layout with all groups and periods in their correct positions.

Can I change the quiz question count mid-session?

No — session length is chosen before you start. Close the quiz and start a new session to pick a different count.

Does the app work offline?

Yes. All element data is bundled with the app and works fully offline. No internet connection is required.

Does the app contain ads?

No. ElementBuddy is completely ad-free.
